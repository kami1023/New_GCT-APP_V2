## 2025-05-15 - Improving Search Discoverability and Accessibility
**Learning:** Adding a keyboard shortcut hint like `[/]` in the search input placeholder significantly improves discoverability for power users, while using declarative React state for filtering (instead of manual DOM manipulation) ensures a more reliable and accessible "empty state" UI.
**Action:** Always include a visual hint for global keyboard shortcuts and prefer declarative state for filtering lists to easily manage empty states and screen reader feedback.
