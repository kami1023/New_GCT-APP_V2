## 2025-05-14 - Search Discoverability & Shortcut Pattern
**Learning:** Adding a global keyboard shortcut (like `/`) significantly improves power-user efficiency, but it must be paired with a visual hint (like `[/]`) in the placeholder to be discoverable by all users. Additionally, ensuring focus is returned to the input after clearing search maintains keyboard flow.
**Action:** Always include shortcut hints in placeholders and manage focus programmatically when clearing inputs.
