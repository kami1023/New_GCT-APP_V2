## 2025-05-15 - [Declarative Search UX]
**Learning:** Imperative DOM manipulation for search logic is fragile and prevents the implementation of robust empty states. Refactoring to declarative React state filtering allows for easy integration of 'No results found' UI and 'Clear Search' actions.
**Action:** Use declarative state-based filtering for all searchable interfaces to ensure UI consistency and support for advanced UX patterns like keyboard shortcuts and empty states.
