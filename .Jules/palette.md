## 2026-05-21 - [Declarative Search & A11y]
**Learning:** Refactoring imperative DOM manipulation to declarative React state not only improves maintainability but also simplifies implementing UX features like empty states and "Clear Search" actions. Adding a keyboard shortcut hint `[/]` in the placeholder significantly improves discoverability for power users.
**Action:** Always prefer state-driven UI for filtering and ensure interactive elements have clear ARIA labels and keyboard access.
